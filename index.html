And it begins

git init => to create the file
hub create projectName => to create a project on github
git add . => to add the current directory to commit
git commit -m "Message for the commit"
git push origin master

Important commands to check on things

git status - to see whats tracked and staged

To clone or not to clone

git clone https://github.com/libgit2/libgit2

  1. creates a directory named "libgit2"
  2. initializes a .git directory inside of it
  3. Pulls down the data for the repository and checks out the latest copy

git clone https://github.com/libgit2/libgit2 mylibgit

  1. Everything above except the target directory is "mylibgit"

All the following work

  1. git clone https://github.com/libgit
  2. git clone git://
  3. git clone user@server:path/to/repo.git
